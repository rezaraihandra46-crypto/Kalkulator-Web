# Kalkulator-Web
A web-based calculator application supporting basic arithmetic operations, unit conversion, and calculation history, built as a team project where I served as Project Leader.
 
# About
Kalkulator.Web is a web-based calculator application developed by a three-member student team as part of an academic project. Beyond standard arithmetic operations, the app includes a unit conversion feature and a calculation history log, allowing users to track and revisit previous calculations.
 
As QA Engineer, evaluated the application's functionality against target system requirements, designed comprehensive test cases covering basic arithmetic, symbolic algebra, and history tracking, and validated database persistence through live functional testing during the team's presentation.
 
# Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Unit conversion
- Calculation history log
- Simple, easy-to-use interface

# Tech Stack
- HTML5
- CSS3
- JavaScript

# My Role — Project Leader
- Executed end-to-end quality assurance protocols across the full software development lifecycle from initial requirement analysis to the final live presentation.

# How to Run
1. Clone or download this repository
2. Open `Menu.html` in your web browser
3. No installation or build steps required
